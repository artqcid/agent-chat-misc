# Agent Chat Misc Extension

A VS Code extension that provides a chat agent with a GUI similar to GitHub Copilot, supporting local and remote LLMs, MCP, RAG, and Embedding integration.

## Features

- Chat interface mimicking GitHub Copilot
- Support for local and remote LLM providers
- MCP, RAG, and Embedding integration
- Configurable system prompts
- Context and prompt management with autocomplete

## Development

1. Install dependencies: `npm install`
2. Compile: `npm run compile`
3. Run in debug mode: Press F5

## Structure

- `src/`: Extension source code
- `webview/`: Webview UI files
- `out/`: Compiled output
